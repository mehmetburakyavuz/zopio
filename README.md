## Overview

[zopio](https://github.com/zopiolabs/zopio) is a modern full-stack business framework designed to build scalable business apps with Auto UI and plugin extensibility ‚Äì built on a developer-first [Next.js](https://nextjs.org/) base. It is designed to be a comprehensive starting point for new apps, providing a solid, opinionated foundation with a minimal amount of configuration.
